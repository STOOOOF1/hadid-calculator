# حاسبة الحديد - ورشة الهياكل المعدنية

تطبيق ويب متكامل لإدارة حسابات وتكاليف ورشة صناعة الهياكل المعدنية. يعمل مباشرة في المتصفح بدون الحاجة إلى سيرفر أو إنترنت.

---

## المميزات

### 🔩 إدارة أنواع الحديد
- إضافة وتعديل وحذف أنواع الحديد (تسليح، زاوية، مربع، صاج...)
- تحديد وحدة القياس لكل نوع (كغم، طن، متر، م²)
- تحديد سعر الوحدة

### 🏗️ إدارة الهياكل
- تعريف أنواع الهياكل (سطحة عادية، هيدروليك، خزانات، هناجر...)
- ربط كل هيكل بالمكونات المطلوبة من الحديد مع الكميات الافتراضية
- تعديل المكونات بسهولة

### 💰 الحقول الإضافية
- إضافة تكاليف إضافية (عمال، إيجار، دهان، نقل، خراطة...)
- نوعين: قيمة ثابتة (ل.س) أو نسبة مئوية (%) من إجمالي الحديد
- تحكم كامل لكل حقل في الحاسبة

### 🧮 الحاسبة (وضعين)

**وضع 1: حسب أنواع الحديد**
- اختر الأصناف من القائمة وحدد الكميات
- احسب التكلفة الكلية مباشرة

**وضع 2: حسب الهيكل**
- اختر الهيكل من القائمة (تظهر مكوناته تلقائياً)
- عدل الكميات حسب الحاجة

لكل حقل إضافي في الحاسبة:
- ✅ **حساب**: يدخل ضمن الحساب والإجمالي الكلي
- 🧾 **فاتورة**: يظهر في الفاتورة المطبوعة
- ❌ **حذف**: إزالة الحقل من القائمة

### 📄 العروض والفاتورة
- حفظ العروض مع اسم العميل والتاريخ
- عرض العروض السابقة والبحث فيها
- فاتورة منسقة للطباعة
- تصنيف العروض: مسودة / نهائي

### 💾 حفظ واسترجاع
- جميع البيانات محفوظة في المتصفح (localStorage)
- تصدير البيانات لملف JSON
- استيراد البيانات من ملف JSON
- إعادة تعيين جميع البيانات

---

## طريقة الاستخدام

1. افتح ملف `index.html` في أي متصفح حديث (Chrome, Firefox, Edge)
2. ابدأ بإضافة أنواع الحديد في تبويب **الحديد**
3. أضف الهياكل في تبويب **الهياكل** مع تحديد مكوناتها من الحديد
4. أضف الحقول الإضافية في تبويب **الحقول الإضافية**
5. استخدم **الحاسبة** لحساب التكاليف:
   - اختر الوضع: حسب الحديد أو حسب الهيكل
   - أضف الأصناف وحدد الكميات
   - اختر الحقول الإضافية المطلوبة
   - شاهد الإجمالي مباشرة
6. احفظ العرض أو اطبع الفاتورة

---

## هيكل البرنامج

الملف الوحيد `index.html` يحتوي على:
- **HTML**: هيكل الصفحات والنماذج
- **CSS**: التصميم الكامل (متجاوب مع الجوال)
- **JavaScript**: كل المنطق (إدارة البيانات، الحسابات، الحفظ، الطباعة)

---

## التقنيات المستخدمة

- HTML5 + CSS3 (مع متغيرات CSS)
- JavaScript ES6+
- localStorage للحفظ في المتصفح
- إخراج طباعة باستخدام `@media print`
- تصميم متجاوب (Responsive) للجوال والتابلت
